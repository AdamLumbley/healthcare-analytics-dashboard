# Healthcare Analytics SQL Project

## Objective
Analyze hospital revenue, billing, and patient trends using SQL.

## Skills Demonstrated
- JOINs
- GROUP BY
- Aggregations
- Conditional aggregates
- Multi-table reporting

## Dataset
Simulated healthcare database including:
- Doctors
- Patients
- Visits (hub table)
- Billing
- Treatments

## Files
- schema.sql → database schema + seed data
- queries.sql → 4 analytical business queries

## Example Questions Answered
1. Total revenue by doctor
2. Total revenue by specialty
3. Patient vs insurance payments by region
4. Revenue by specialty and region
